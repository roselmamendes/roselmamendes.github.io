Preparing the local environment

I am using Docker. So first I created a Dockerfile as following:

FROM python:latest

WORKDIR /usr/src/app

COPY requirements.txt ./
RUN pip install --no-cache-dir -r requirements.txt

COPY . .

EXPOSE 8000
